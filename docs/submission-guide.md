## DoraHacks Submission Steps
1. Go to https://dorahacks.io/hackathon/qvac-unleach-edge-ai-i
2. Click "Submit Project" or "BUIDL"
3. Fill in project name, description, and track
4. Add all team members to the project page
5. Paste GitHub repo URL
6. Upload YouTube demo video link (set to "Unlisted")
7. Upload device specs and API disclosure files
8. Submit before deadline
